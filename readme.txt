Adds a Dynamic Tag to Elementor Pro for Restrict Content Pro to return whether the current user has an active membership or not.

== Description ==

This plugin creates a dynamic tag for Elementor Pro that returns whether or not the user has an active membership or not. Used in conjunction with the Dynamic Conditions https://wordpress.org/plugins/dynamicconditions/ plugin you can easily use RCP conditional logic in Elementor Pro.

== Installation ==

You need

**Requires Elementor Pro and Restrict Content Pro to work**

1. Unzip `rcp-elementor-dynamic-tags.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
